Ansible Playbook
--------------------
we write everything in ansible is called ansible playbooks

ansible adhoc commands
----------------------
to run ansible adhoc commands 
inventory
--------------------------
it will store all the server ip addresses

ansible -i inventory file location <ip address>

ansible -i inventory all -m "shell" -a "touch devops class" -->it is ansible adhoc commands

-m --> modules
-a --> what is the command that is used to run

Playbook
---------------------
vim first-playbook.yaml

ex:create a nginx server and restart the nginx server

-name: Install and start Nginx
 hosts:all
 become:root
 
 tasks:
  - name: install nginx
    apt:
	 name: nginx
	 state: present
  - name: start nginx
    shell: systemctl start nginx
	       or
    service: 
	 name: nginx
	 state: started
	 
save the file

execute the playbook by using ansible-playbook -i inventory first-playbook.yaml
